
代码仓库创建说明
====

* 代码仓库只能由管理人员创建，管理人员给每一个仓库添加开发人员，并复制该邀请链接给相应的开发人员
* 开发人员登录自己的github后，在浏览器打开邀请链接，接受邀请

开发人员一些操作指南
====

* 本地安装好git
* 打开cmd窗口，git clone https://相应的仓库
* cd 相应的目录
* git pull          ---- 用来把服务器的仓库同步到本地
* git add  *        ---- 把本地所有的变动加入操作缓存
* git commit -m‘一些注释’  ---提交
* git push                ----把缓存提交到服务器上
